# Alif-Flow Sales System

Alif-Flow is a modern, professional sales tracking and reporting mobile application designed for small businesses. It streamlines the process of weekly sales data entry for sellers and provides a robust review and approval workflow for managers and admins.

## Project Goal

The primary objective of Alif-Flow is to provide a clean, fast, and optimized mobile interface for daily sales reporting with minimal friction.

## Key Features

- **Seller Dashboard:** Fast data entry for weekly sales and inventory, including product movement tracking.
- **Admin Dashboard:** Centralized view for managers to review, approve, or reject seller reports with clear status tracking.
- **Authentication:** Secure login and registration with role-based access control.
- **Modern UI:** Built using the "Luminous Sales Logic" design system, featuring a SaaS-inspired aesthetic with deep cyan accents, clean typography (Inter), and touch-friendly components.

## Tech Stack

- **Framework:** [Flutter](https://flutter.dev/)
- **Language:** [Dart](https://dart.dev/)
- **Design System:** Luminous Sales Logic (Modern Dark/Light mode, 8px/12px radius standards)
- **Target Platform:** Android

## Development Status

This project is currently in the UI implementation phase, with core screens and navigation established.

## Getting Started

To run this project locally:

1. Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Connect an Android device or emulator.
5. Execute `flutter run`.
